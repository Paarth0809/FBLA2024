    JavaScript Email Library

    SMTP.js provides a simple and effective way to send emails directly from JavaScript without requiring server-side code. This can be particularly useful for web applications that need to send notifications or contact forms.

    Email Marketing Tools

    Elastic Email provides robust tools for managing and sending email campaigns. We used Elasticemail as the middle man in sending emails for both the Vaccine Scheduling and the Job application. Our Elasticemail account also allows us to see when the email was sent, the contents of the email, and when the user opens the email.

    Visual Studio Code

    Visual Studio Code (VS Code) is a powerful and versatile code editor developed by Microsoft. It supports a wide range of programming languages and provides a wide variety of features and integrations. Its extensive marketplace offers numerous extensions to enhance functionality. VS Code helped the development of our website as it was our primary code editor and allowed us to connect the HTML, CSS and JS aspects of our websites to create the perfect website for our client Futura Medical.

    ChatGPT and OpenAI API Key

    ChatGPT, developed by OpenAI, is a cutting-edge conversational AI model that can understand and generate human-like text. By using the OpenAI API, we were able to integrate ChatGPT into our website and build our client, Futura, their own chatbot, which allows users to get answers to specific questions they may have about the company. The API Key allows us to connect our chatbot to the Open AI LLM (Large Language Model). On top of this we provided Open AI with custom data abiout Futura creating a personalized chatbot.
